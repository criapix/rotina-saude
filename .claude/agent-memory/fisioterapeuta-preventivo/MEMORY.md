### Padrões Identificados
- **Rotura parcial do manguito:** Protocolo excêntrico é ESSENCIAL — não pode ser replicado em academia
- **Modic I cervical:** Tração mecânica + terapia manual são insubstituíveis
- **Hipomobilidade torácica:** Máquina de rotação de tronco é EXCELENTE substituto terapêutico (até melhor que cabo para D3-D10)
- **Double crush syndrome:** Ergonomia MTB (luvas, manoplas, pausas) + ENMG são fundamentais
- **Substitutos de core:** Máquinas são SUPERIORES a cabos para este caso (protegem ombro + isolam core)

### Estratégias que Funcionaram
- Protocolo de ativação pré-treino A e C → mantém manguito preparado
- Supino máquina com cotovelos 45° → protege supraespinhal vs supino livre
- Protocolo matinal domiciliar → mobilidade cervical/torácica diária
- Gelo após treino upper → controle bursite
- Exercícios com apoio de tronco → proteção cervical

### Lições Aprendidas
- Série de academia bem estruturada pode COMPENSAR PARCIALMENTE fisioterapia, mas NUNCA SUBSTITUIR completamente
- Protocolo excêntrico é o diferencial terapêutico que não pode ser replicado em equipamentos convencionais
- Dissociação cervical/torácica (olhar fixo à frente) é CRÍTICA em exercícios de rotação torácica
- Exercícios terapêuticos (rotação externa, face pull) devem progredir por VOLUME/CONTROLE, nunca por carga
- Máquinas são superiores a cabos para pacientes com lesões de ombro (eliminam preensão ativa + estabilizam tronco)
