## Restrições Confirmadas

### 1. Rotura parcial do manguito rotador
- Impacta exercícios de ombro e tração supina
- Para nutrição: garantir ingestão adequada de proteína para suporte ao tecido conectivo
- Micronutrientes relevantes: vitamina C, colágeno hidrolisado, ômega-3 (anti-inflamatório)
- Evitar déficit calórico muito agressivo — pode comprometer recuperação tecidual

### 2. Coluna cervicotorácica degenerativa
- Impacta exercícios de alta carga axial
- Para nutrição: ômega-3, vitamina D e magnésio têm papel anti-inflamatório relevante
- Hidratação adequada importante para saúde do disco intervertebral

## Implicações para estratégia de perda de gordura
- Déficit calórico deve ser moderado (não agressivo) para não comprometer recuperação das lesões
- Suporte nutricional anti-inflamatório deve ser prioritário
